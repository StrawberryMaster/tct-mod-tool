  This file demonstrates how to use Vue 3 directly instead of Vue 2 + compatibility layer.
  To enable this, replace the Vue 2 script imports in index.html with the following:

  Replace:
    <script src="./js/vue.js"></script>
    <script src="./js/vue3-compat.js"></script>

  With:
    <script src="./js/vue3.js"></script>

  The vue3-modernizer.js will ensure component registration works properly.
-->
